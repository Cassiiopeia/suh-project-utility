SUH Random 닉네임 생성기 가이드

기능 개요

SUH Random은 약 96만 가지 조합의 개성 있는 닉네임을 자동으로 생성하는 도구입니다. 형용사와 명사를 조합하여 멋진고양이, 빛나는호랑이와 같은 자연스러운 닉네임을 만들어줍니다. 이 기능은 suh-random-engine이라는 독립 자바 라이브러리를 기반으로 구현되었으며, 해당 라이브러리는 Nexus 저장소에서 배포됩니다.

접속 방법

대시보드에서 Dev Tools 섹션의 SUH 랜덤 엔진 카드를 클릭합니다. 또는 헤더 메뉴에서 유틸리티를 클릭한 후 SUH 랜덤을 선택합니다. 직접 URL로 접속할 경우 lab.suhsaechan.kr/suh-random 주소를 사용합니다.

사용 방법

닉네임 생성 옵션 카드에서 원하는 설정을 선택합니다. 먼저 언어를 선택합니다. 한국어와 영어 중에서 선택할 수 있으며 기본값은 한국어입니다.

다음으로 닉네임 타입을 선택합니다. 기본 닉네임은 형용사와 명사를 조합한 형태로 멋진고양이와 같이 생성됩니다. 숫자 접미사는 닉네임 뒤에 숫자를 붙여 멋진고양이-1234 형태로 생성됩니다. UUID 접미사는 닉네임 뒤에 UUID를 붙여 멋진고양이-abcd 형태로 생성됩니다.

정치인 닉네임 카테고리도 있습니다. 정치인 닉네임은 형용사와 정치인 이름을 조합하여 멋진윤석열과 같이 생성됩니다. 정치인 숫자는 정치인 닉네임에 숫자를 붙이고, 정치인 UUID는 정치인 닉네임에 UUID를 붙입니다.

성인용 닉네임 카테고리는 만 19세 이상만 사용할 수 있습니다. 성인용 형용사와 명사를 조합하며, 숫자나 UUID를 붙이는 옵션도 있습니다.

숫자 접미사나 UUID 접미사 타입을 선택하면 길이 설정 옵션이 나타납니다. 숫자 길이는 1부터 10까지 설정할 수 있으며 기본값은 4입니다. UUID 길이도 동일하게 설정할 수 있습니다.

성인용 닉네임을 선택하면 동의 사항이 표시됩니다. 만 19세 이상임을 확인하는 체크박스, 성인용 콘텐츠 사용에 대한 책임 동의 체크박스, 불법적인 용도로 사용하지 않을 것을 동의하는 체크박스 세 가지 모두 체크해야 합니다.

설정을 완료한 후 닉네임 생성하기 버튼을 클릭합니다.

결과 확인

생성 결과 카드에서 결과를 확인할 수 있습니다. 결과 탭에서는 생성된 닉네임이 크게 표시되고 복사하기 버튼으로 클립보드에 복사할 수 있습니다. 아래에는 닉네임 타입, 언어, 숫자 길이, UUID 길이, 생성 시간 정보가 테이블로 표시됩니다.

JSON 탭에서는 API 응답 전체를 JSON 형식으로 확인할 수 있습니다. 우측 상단의 복사 버튼으로 JSON 전체를 복사할 수 있습니다.

히스토리 탭에서는 최근에 생성한 닉네임 10개를 확인할 수 있습니다. 각 항목 옆의 복사 버튼으로 해당 닉네임을 복사할 수 있습니다. 히스토리는 브라우저 로컬 스토리지에 저장되므로 브라우저를 닫아도 유지됩니다.

기술 구현

이 기능은 suh-random-engine 라이브러리 버전 1.1.0을 사용합니다. 해당 라이브러리는 Java 17 이상과 Spring Boot 3.x 환경에서 동작합니다.

API 엔드포인트는 POST /api/suh-random/generate 입니다. 요청 파라미터로 nicknameType, locale, numberLength, uuidLength, isAdultConsent를 받습니다. 응답에는 nickname, nicknameType, locale, numberLength, uuidLength, generatedAt 필드가 포함됩니다.

닉네임 생성 시 SuhRandomKit 빌더를 사용하여 설정을 구성합니다. locale으로 언어를 설정하고, numberLength와 uuidLength로 접미사 길이를 설정하고, enableAdultContent로 성인용 콘텐츠 활성화 여부를 설정합니다.

내부 동작

RandomNicknameController가 API 요청을 처리합니다. 입력 파라미터의 유효성을 검사하고 기본값을 설정합니다. nicknameType이 MATURE로 시작하는 경우 성인 동의 여부를 확인합니다. 동의하지 않으면 IllegalArgumentException을 발생시킵니다.

SuhRandomKit 인스턴스를 생성하고 nicknameType에 따라 적절한 메서드를 호출합니다. SIMPLE은 simpleNickname 메서드를, NUMBER는 nicknameWithNumber 메서드를, UUID는 nicknameWithUuid 메서드를 호출합니다. 정치인 타입은 politicianNickname 계열 메서드를, 성인용 타입은 matureNickname 계열 메서드를 호출합니다.

닉네임 생성 후 StatisticsService의 logFeatureUsageAsync 메서드를 호출하여 사용 통계를 기록합니다. FeatureType.SUH_RANDOM으로 분류되어 대시보드의 기능별 상세 통계에 표시됩니다.

외부 프로젝트

suh-random-engine은 독립적인 자바 라이브러리로 GitHub에서 소스 코드를 확인할 수 있습니다. 저장소 주소는 github.com/Cassiiopeia/suh-random-engine 입니다. MIT 라이선스로 배포되며 누구나 자유롭게 사용할 수 있습니다.

라이브러리는 Nexus 저장소에서 배포됩니다. Gradle에서 implementation 'me.suhsaechan:suh-random-engine:1.1.0' 의존성을 추가하여 사용할 수 있습니다. Nexus 저장소 주소는 nexus.suhsaechan.kr 입니다.

닉네임 타입 요약

SIMPLE 타입은 형용사와 명사를 조합한 기본 형태입니다. 예시로 멋진고양이, 빛나는호랑이가 있습니다.

NUMBER 타입은 기본 닉네임에 숫자를 붙인 형태입니다. 예시로 멋진고양이-1234가 있습니다.

UUID 타입은 기본 닉네임에 UUID를 붙인 형태입니다. 예시로 멋진고양이-abcd가 있습니다.

POLITICIAN 타입은 형용사와 정치인 이름을 조합한 형태입니다. 예시로 멋진윤석열이 있습니다.

POLITICIAN_NUMBER 타입은 정치인 닉네임에 숫자를 붙인 형태입니다. 예시로 빛나는이재명-1234가 있습니다.

POLITICIAN_UUID 타입은 정치인 닉네임에 UUID를 붙인 형태입니다. 예시로 멋진이준석-abcd가 있습니다.

MATURE 타입은 성인용 형용사와 명사를 조합한 형태입니다. 만 19세 이상만 사용할 수 있습니다.

MATURE_NUMBER 타입은 성인용 닉네임에 숫자를 붙인 형태입니다. 만 19세 이상만 사용할 수 있습니다.

MATURE_UUID 타입은 성인용 닉네임에 UUID를 붙인 형태입니다. 만 19세 이상만 사용할 수 있습니다.

주의사항

성인용 콘텐츠는 만 19세 이상만 사용할 수 있습니다. 사용 시 세 가지 동의 사항에 모두 체크해야 합니다. 성인용 콘텐츠의 부적절한 사용으로 인한 모든 법적 책임은 사용자 본인에게 있으며 서비스 제공자는 어떠한 책임도 지지 않습니다.

생성된 닉네임을 불법적인 용도로 사용하지 않아야 합니다. 특히 정치인 닉네임의 경우 명예훼손이나 비방 목적으로 사용하면 법적 책임이 따를 수 있습니다.
