# 사이트 기능 소개

> **lab.suhsaechan.kr**에 오신 것을 환영합니다!  
> 이 사이트는 개인 실험실로, 다양한 유틸리티 도구들을 제공합니다.

---

## 🎯 이 사이트는 무엇인가요?

**SUH-PROJECT-UTILITY**는 개발자가 필요할 때마다 만든 실험적인 유틸리티 모음입니다.

- **개인용으로 만들어졌지만, 누구나 사용할 수 있습니다**
- **유용하면 남기고, 아니면 좋은 실험이었습니다**
- **복잡한 프레임워크 없이, 그냥 작동하는 것들입니다**

---

## 🛠️ 주요 기능

### 1. 🌱 Grass Planter (잔디 심기)

**GitHub 잔디를 자동으로 채워주는 도구**

- GitHub 프로필의 Contribution Graph를 자동으로 채웁니다
- 프로필을 생성하고 자동 커밋을 설정할 수 있습니다
- 수동 커밋 실행 및 커밋 로그 조회 기능 제공
- 특정 날짜의 기여도 레벨 확인 가능

**사용 시나리오**:  
GitHub 잔디를 꾸준히 채우고 싶지만 매일 커밋하기 어려울 때, 자동으로 커밋을 생성해줍니다.

---

### 2. 🤖 AI Translator (AI 번역기)

**OpenAI를 활용한 커스텀 번역 서비스**

- 다양한 언어 쌍 지원
- OpenAI의 고급 번역 모델 활용
- 실시간 번역 제공

**사용 시나리오**:  
일반 번역기로는 부족한 전문 용어나 문맥을 고려한 번역이 필요할 때 사용합니다.

---

### 3. 📝 Issue Helper (이슈 헬퍼)

**GitHub 이슈에서 브랜치명과 커밋 메시지 자동 생성**

- GitHub 이슈 제목/내용을 분석하여 브랜치명 생성
- 커밋 메시지 자동 생성
- GitHub Workflow와 연동 가능

**사용 시나리오**:  
GitHub 이슈를 만들 때마다 브랜치명과 커밋 메시지를 일관되게 만들고 싶을 때 사용합니다.

---

### 4. 📚 Study Plan (학습 관리)

**마크다운 기반 학습 노트 관리 시스템**

- 카테고리별 학습 포스트 관리
- 마크다운 에디터로 스타일 있는 노트 작성
- 첨부 파일 업로드 지원
- 최근 포스트, 인기 포스트 조회 기능
- 검색 기능 제공

**사용 시나리오**:  
개인 학습 내용을 체계적으로 정리하고 관리하고 싶을 때 사용합니다.

---

### 5. 🐳 Docker Logs (도커 로그 모니터링)

**실시간 Docker 컨테이너 로그 모니터링**

- 실행 중인 컨테이너 목록 조회
- 실시간 로그 폴링 (Polling)
- 여러 컨테이너 동시 모니터링 가능
- 로그 화면 초기화 기능

**사용 시나리오**:  
서버에서 실행 중인 Docker 컨테이너의 로그를 웹에서 쉽게 확인하고 싶을 때 사용합니다.

---

### 6. 💬 Chatbot (AI 챗봇)

**벡터 검색 기반 지식 챗봇**

- 문서 기반 RAG (Retrieval-Augmented Generation) 구현
- 실시간 스트리밍 응답 (SSE)
- 대화 이력 관리
- 메시지 피드백 (좋아요/싫어요) 기능
- 문서 관리 기능 (관리자용)

**사용 시나리오**:  
사이트에 대한 질문이나 도움말을 AI 챗봇에게 물어볼 때 사용합니다.

---

### 7. 🚌 Somansa Bus (소만사 버스 예약)

**버스 예약 및 관리 시스템**

- 사용자 등록 및 관리
- 버스 노선 관리
- 스케줄 생성 및 관리
- 수동 예약 기능
- 예약 이력 조회

**사용 시나리오**:  
회사나 조직 내부의 버스 예약 시스템이 필요할 때 사용합니다.

---

### 8. 📢 Notice (공지사항)

**공지사항 및 댓글 시스템**

- 공지사항 작성, 수정, 삭제
- 공지사항 활성화/비활성화
- 댓글 기능
- 검색 기능

**사용 시나리오**:  
사이트의 중요한 공지사항을 사용자들에게 전달하고 싶을 때 사용합니다.

---

### 9. 📊 Dashboard (대시보드)

**사이트 통계 및 현황 대시보드**

- 방문자 통계
- 챗봇 사용 통계
- 토큰 사용량 통계
- 콘텐츠 조회수 통계
- 통합 요약 정보 제공 (5분 캐싱)

**사용 시나리오**:  
사이트의 전반적인 사용 현황을 한눈에 파악하고 싶을 때 확인합니다.

---

## 🚀 기술 스택

```
Backend    → Spring Boot 3.4 + Java 17
Database   → PostgreSQL + Redis
Frontend   → Thymeleaf + TailwindCSS + DaisyUI
Deploy     → Docker + GitHub Actions + Synology NAS
```

## 💡 사용 팁

1. **대부분의 기능은 로그인 없이 사용 가능합니다**
   - 번역, 챗봇, 이슈 헬퍼 등은 인증 없이 사용할 수 있습니다

2. **관리자 기능**
   - 일부 기능(문서 관리, 통계 등)은 관리자 권한이 필요합니다

3. **API 사용**
   - 모든 기능은 REST API로도 제공되며, Swagger UI에서 확인할 수 있습니다
