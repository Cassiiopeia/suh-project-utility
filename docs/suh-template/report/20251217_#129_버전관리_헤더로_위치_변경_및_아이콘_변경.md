### 📌 작업 개요
버전 태그를 헤더로 이동하여 로그인 후에도 항상 표시되도록 개선. 반응형 디자인 및 아이콘 변경 포함

**보고서 파일**: `.report/20251217_#129_버전관리_헤더로_위치_변경_및_아이콘_변경.md`

### 🎯 구현 목표
- 버전 태그를 헤더의 "SUHSAECHAN LAB" 오른쪽에 배치하여 로그인 후에도 항상 표시
- 버전 태그 아이콘을 태그 아이콘으로 변경
- 로그인 페이지의 About Me 아이콘을 커서 아이콘으로 변경하여 클릭 가능한 UI임을 명확히 표시
- 버전 태그 색상을 보라색에서 하늘색으로 변경
- 반응형 디자인 개선: 모바일에서 "SUHSAECHAN LAB"을 "S.LAB"으로 표시
- 1024px 이하 화면에서 버전 태그는 아이콘만 표시
- 로그인 페이지 form-control 간격 조정

### ✅ 구현 내용

#### 버전 태그 헤더 배치
- **파일**: `Suh-Web/src/main/resources/templates/fragments/header.html`
- **변경 내용**: 버전 태그를 헤더의 navbar-start 영역에 배치. "SUHSAECHAN LAB" 텍스트 오른쪽에 위치
- **이유**: 로그인 전후 모두에서 버전 정보를 확인할 수 있도록 항상 표시되는 위치로 이동

#### 버전 태그 아이콘 변경
- **파일**: `Suh-Web/src/main/resources/templates/fragments/header.html`, `Suh-Web/src/main/resources/templates/pages/login.html`
- **변경 내용**: 버전 태그 아이콘을 `fa-tag`로 변경
- **이유**: 버전 태그의 의미를 더 명확하게 전달

#### About Me 아이콘 변경 (로그인 페이지)
- **파일**: `Suh-Web/src/main/resources/templates/pages/login.html`
- **변경 내용**: About Me 배지의 아이콘을 `fa-user`에서 `fa-arrow-pointer`로 변경
- **이유**: 클릭 가능한 UI임을 시각적으로 명확히 표시

#### 버전 태그 색상 변경
- **파일**: `Suh-Web/src/main/resources/static/css/common.css`
- **변경 내용**: 버전 배지 색상을 보라색에서 하늘색(cyan 계열)으로 변경
  - 라이트 모드: 배경 `#e0f2fe`, 텍스트 `#0369a1`, 테두리 `#bae6fd`
  - 다크 모드: 배경 `#0c4a6e`, 텍스트 `#7dd3fc`, 테두리 `#075985`
- **이유**: 디자인 일관성 및 가독성 개선

#### 반응형 텍스트 표시
- **파일**: `Suh-Web/src/main/resources/templates/fragments/header.html`
- **변경 내용**: 
  - 모바일(`lg:hidden`): "S.LAB" 표시
  - 데스크탑(`hidden lg:inline`): "SUHSAECHAN LAB" 표시
- **이유**: 모바일 화면에서 텍스트가 두 줄로 쪼개지는 문제 해결

#### 반응형 버전 태그 표시
- **파일**: `Suh-Web/src/main/resources/templates/fragments/header.html`
- **변경 내용**: 
  - 1024px 이하(`lg` 브레이크포인트 미만): 태그 아이콘만 표시
  - 1024px 이상: 태그 아이콘 + 버전 번호 표시
- **이유**: 작은 화면에서 공간 효율성 확보

#### 로그인 페이지 form-control 간격 조정
- **파일**: `Suh-Web/src/main/resources/templates/pages/login.html`
- **변경 내용**: 아이디 입력 필드에 `-mb-2` 클래스 추가하여 아이디와 비밀번호 필드 간 간격 축소
- **이유**: form-control의 기본 하단 마진이 커서 간격이 과도하게 넓었던 문제 해결

### 🔧 주요 변경사항 상세

#### header.html - 버전 태그 배치
버전 태그를 헤더의 navbar-start 영역에 배치하여 로고와 텍스트 바로 옆에 위치시킴
```html
<!-- 버전 태그 (모바일: 아이콘만, 데스크탑: 아이콘 + 버전) -->
<button onclick="openChangelogModal()"
        class="badge gap-0.5 ml-3 px-1.5 py-0.5 lg:px-2 lg:py-1 lg:gap-1 hover:opacity-80 transition-opacity cursor-pointer ml-1 version-badge text-[0.65rem]"
        title="버전 정보">
  <i class="fa-solid fa-tag text-[0.7rem]"></i>
  <!-- 데스크탑에서만 버전 숫자 표시 -->
  <span id="header-version-tag" class="version-tag-text hidden lg:inline">v1.0.0</span>
</button>
```

**특이사항**:
- `lg:inline` 클래스로 1024px 이상에서만 버전 번호 표시
- `version-badge` 클래스로 하늘색 스타일 적용
- `openChangelogModal()` 함수로 클릭 시 changelog 모달 열기

#### common.css - 버전 배지 스타일
하늘색 계열 색상으로 버전 배지 스타일 정의
```css
.version-badge {
  background-color: #e0f2fe !important;
  color: #0369a1 !important;
  border: 1px solid #bae6fd !important;
}

[data-theme="dark"] .version-badge {
  background-color: #0c4a6e !important;
  color: #7dd3fc !important;
  border: 1px solid #075985 !important;
}
```

**특이사항**:
- 라이트/다크 모드 모두 지원
- `!important` 플래그로 다른 스타일 오버라이드 보장

#### login.html - About Me 아이콘 및 form-control 간격
About Me 배지 아이콘을 커서 아이콘으로 변경하고, form-control 간격 조정
```html
<a href="/profile" class="badge badge-soft badge-primary gap-1 px-3 py-1 text-xs hover:opacity-80 transition-opacity cursor-pointer">
  <i class="fa-solid fa-arrow-pointer"></i>
  ABOUT ME
</a>

<!-- 아이디 입력 -->
<div class="form-control -mb-2">
```

**특이사항**:
- `-mb-2` 클래스로 음수 마진 적용하여 간격 축소
- 커서 아이콘으로 클릭 가능성 시각적 표시

### 📌 참고사항
- 버전 태그는 헤더와 로그인 페이지 양쪽에 모두 표시됨
- 헤더의 버전 태그는 `header-version-tag` ID를 사용하여 JavaScript로 동적 업데이트 가능
- Tailwind의 `lg` 브레이크포인트(1024px)를 기준으로 반응형 동작
- 버전 정보는 `common.js`의 `loadVersionFromChangelog()` 함수로 CHANGELOG.json에서 자동 로드됨
