### 📌 작업 개요
대시보드에 서브도메인 바로가기 섹션 추가 및 Projects 섹션 전면 재구조화.
Server/Services 분리, 모든 프로젝트 카드 클릭 가능 패턴 적용, 신규 프로젝트 "Re: WAVE" 추가.

**관련 이슈**: [#143](https://github.com/Cassiiopeia/suh-project-utility/issues/143)

---

### 🎯 구현 목표
- 서브도메인을 직접 URL 입력 없이 클릭 한 번으로 접근
- Server(시놀로지)와 Services(인프라) 섹션 분리
- Projects 섹션 카드를 클릭 가능한 패턴으로 통일
- 다크모드 지원

---

### ✅ 구현 내용

#### 1. Server / Services 섹션 분리
- **파일**: `Suh-Web/src/main/resources/templates/pages/dashboard.html`
- **변경 내용**: 기존 "Server & Services" 통합 섹션을 두 개의 독립 섹션으로 분리
  - **Server**: 시놀로지 NAS 관련 (DSM, Note Station)
  - **Services**: 인프라 서비스 (Nexus, Qdrant, RabbitMQ, Traefik)

#### 2. Projects 섹션 전면 재구조화
- **파일**: `Suh-Web/src/main/resources/templates/pages/dashboard.html`
- **변경 내용**:
  - Mapsy 프로젝트를 Projects 섹션 최상단에 추가
  - 모든 카드를 클릭 가능한 `<a>` 태그 패턴으로 변경
  - "활성" 배지 모두 제거
  - Figma 카드 제목을 "Figma"로 통일
  - RomRom: Figma 카드 1개만 유지
  - 신규 프로젝트 "Re: WAVE" 추가

#### 3. 다크모드 CSS 스타일 추가
- **파일**: `Suh-Web/src/main/resources/static/css/common.css`
- **변경 내용**:
  - `bg-violet-50`, `bg-violet-100` 다크모드 배경색 추가
  - 기존 다크모드 스타일 유지

#### 클릭 가능한 카드 패턴

```html
<a href="https://..." target="_blank" rel="noopener noreferrer"
   class="card bg-base-100 border border-gray-100 shadow-sm hover:shadow-lg hover:-translate-y-1 transition-all duration-200">
  <div class="card-body p-5">
    <div class="flex items-center gap-3">
      <div class="w-12 h-12 rounded-lg bg-{color}-100 flex items-center justify-center">
        <i class="fa-solid fa-{icon} text-{color}-600 text-xl"></i>
      </div>
      <div>
        <h3 class="font-bold text-gray-900">카드명</h3>
        <span class="badge badge-soft badge-{type} badge-xs">배지</span>
      </div>
    </div>
    <p class="text-gray-500 text-sm mt-3">설명</p>
  </div>
</a>
```

**특이사항**:
- 모든 외부 링크에 `target="_blank"` + `rel="noopener noreferrer"` 적용
- 카드 전체가 `<a>` 태그로 감싸져 클릭 영역 확장
- 기존 버튼 방식에서 카드 전체 클릭 방식으로 변경
