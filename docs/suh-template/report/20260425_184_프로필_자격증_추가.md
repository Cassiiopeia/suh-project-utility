# [기능추가][프로필] 자격증 정보 추가 (리눅스마스터2급, SQLD)

## 개요
대시보드 프로필의 Certifications 섹션에 리눅스마스터 2급과 SQLD 자격증을 신규 추가하고, 자격증 카드 클릭 시 전체화면 오버레이로 자격증 이미지를 확인할 수 있는 기능을 구현했다.

## 변경 사항

### UI 템플릿
- `Suh-Web/src/main/resources/templates/pages/dashboard.html`: Certifications 섹션에 리눅스마스터 2급(KAIT), SQLD(K-data) 카드 추가. 기존 카드(정보처리기사, 화학분석기사)도 클릭 인터랙션 적용. 자격증 전체화면 오버레이 HTML 및 `toggleCert`, `closeCert` JavaScript 함수 추가. ESC 키로 오버레이 닫기 지원.

### CSS
- `Suh-Web/src/main/resources/static/css/common.css`: `.cert-overlay`(전체화면 블러 배경), `.cert-overlay-card`(최대 420px 카드) 클래스 추가.

### 이미지 에셋
- `Suh-Web/src/main/resources/static/images/certificates/cert-linux-master.png`: 리눅스마스터 2급 자격증 이미지
- `Suh-Web/src/main/resources/static/images/certificates/cert-sqld.png`: SQLD 자격증 이미지
- `Suh-Web/src/main/resources/static/images/certificates/cert-info-processing.png`: 정보처리기사 자격증 이미지
- `Suh-Web/src/main/resources/static/images/certificates/cert-chemical.png`: 화학분석기사 자격증 이미지
- `Suh-Web/src/main/resources/static/images/kaitLogo.png`: KAIT 기관 로고
- `Suh-Web/src/main/resources/static/images/kdataLogo.png`: K-data 기관 로고

## 주요 구현 내용
자격증 카드에 `onclick="toggleCert(imgSrc, imgAlt, card)"` 이벤트를 연결하여, 같은 카드를 다시 클릭하면 오버레이가 닫히는 토글 방식으로 구현했다. 오버레이 배경 클릭, ESC 키로도 닫을 수 있다. `.cert-overlay`는 `position: fixed; inset: 0`으로 전체화면을 덮으며, `backdrop-filter: blur(4px)`로 배경을 블러 처리한다. 자격증 카드는 `hover-3d` 클래스를 적용하여 기존 인터랙티브 카드 디자인과 일관성을 유지했다.

## 주의사항
- 자격증 이미지 파일은 바이너리로 커밋되었으므로, 향후 이미지 교체 시 동일 경로에 덮어쓰면 반영된다.
- 오버레이 내 이미지는 `object-contain`으로 비율을 유지하므로 이미지 크기와 무관하게 정상 표시된다.
