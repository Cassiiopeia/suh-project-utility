# [기능개선][대시보드] 인프라 서비스 바로가기 카드 추가 (Cloudflare, pgAdmin, MongoDB, MinIO)

## 개요
대시보드의 인프라 서비스 바로가기 섹션에 Cloudflare, pgAdmin, MongoDB Express, MinIO 4개의 카드를 추가했다. 기존에 Nexus, Portainer 등만 있던 인프라 링크 모음을 확장하여 운영 중인 주요 서비스에 빠르게 접근할 수 있도록 개선했다.

## 변경 사항

### UI 템플릿
- `Suh-Web/src/main/resources/templates/pages/dashboard.html`: 인프라 서비스 그리드에 카드 4개 추가.
  - Cloudflare(`dash.cloudflare.com`): DNS/CDN 관리. 주황색 아이콘(`fa-cloud`), `badge-warning`.
  - pgAdmin(`pgadmin.suhsaechan.kr`): PostgreSQL 관리 콘솔. 파란색 아이콘(`fa-elephant`), `badge-info`.
  - MongoDB Express(`mongo.suhsaechan.kr`): MongoDB 관리 콘솔. 초록색 아이콘(`fa-leaf`), `badge-success`.
  - MinIO(`minio.suhsaechan.kr`): S3 호환 오브젝트 스토리지. 빨간색 아이콘(`fa-hard-drive`), `badge-error`.

## 주요 구현 내용
기존 인프라 카드와 동일한 구조(`card bg-base-100 border shadow-sm hover:shadow-lg hover:-translate-y-1 transition-all`)를 재사용하여 UI 일관성을 유지했다. 각 서비스는 `target="_blank" rel="noopener noreferrer"`로 새 탭에서 열린다. 서비스 특성에 맞는 색상 코딩을 적용하여 시각적 구분이 용이하다.

## 주의사항
- Cloudflare 링크에 계정 ID가 URL에 포함되어 있으나, 대시보드는 관리자 전용 페이지이므로 허용 범위로 판단했다.
- 각 서비스 URL 변경 시 템플릿의 `href` 속성을 직접 수정해야 한다.
