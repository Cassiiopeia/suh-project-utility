# 트립게더 앱 링크/다운로드 버튼 및 Projects 그룹 추가

## 개요
대시보드(`dashboard.html`)에서 트립게더 접근성을 개선했다. Team Projects 섹션의 트립게더 카드를 실제 서비스 앱(`tripgether.app`)으로 연결하고 앱스토어 다운로드 버튼을 추가했으며, Projects 섹션에 트립게더 관련 리소스를 한곳에 모은 Tripgether 그룹 박스를 신설했다.

## 변경 사항

### Team Projects — 트립게더 카드
- `Suh-Web/src/main/resources/templates/pages/dashboard.html`
  - 카드 클릭 링크를 GitHub Org(`github.com/TEAM-Tripgether`) → 앱 메인(`https://tripgether.app/`)으로 변경
  - 하단 액션 버튼을 2개(GitHub, Instagram) → 4개로 확장
    - iOS: `https://apps.apple.com/app/tripgether/id6746592757`
    - Android: `https://play.google.com/store/apps/details?id=app.tripgether`

### Projects 섹션 — Tripgether 그룹 신설
- `Suh-Web/src/main/resources/templates/pages/dashboard.html`
  - Mapsy 그룹 박스 바로 위에 Tripgether 그룹 박스 추가 (cyan 계열 색상)
  - 서브카드 4종 배치
    - 앱 메인: `https://tripgether.app/`
    - BE 레포: `https://github.com/Cassiiopeia/Tripgether-BE`
    - FE 레포: `https://github.com/Cassiiopeia/tripgether-FE`
    - AI 레포: `https://github.com/TEAM-Tripgether/tripgether-python` (Private 배지)

## 주요 구현 내용
- 기존 카드/그룹 박스 마크업 패턴을 그대로 재사용하여 일관성 유지
- CSP 정책 준수를 위해 인라인 `style` 속성 미사용, 모든 스타일은 기존 클래스로 처리
- Tailwind 표준 유틸리티 클래스만 사용 (하드코딩 대괄호 값 없음)
- AI 레포는 비공개 저장소이므로 `Private` 배지로 명시 표기

## 주의사항
- AI 레포(`tripgether-python`)는 private이므로 접근 권한이 없는 방문자는 GitHub 404 페이지를 보게 된다. 의도된 동작이며 배지로 비공개임을 안내한다.
- 내부망 환경 특성상 서버 직접 실행 검증은 미수행. 브라우저 렌더링 확인은 별도 환경에서 필요하다.
