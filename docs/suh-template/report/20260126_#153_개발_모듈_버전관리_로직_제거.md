### 📌 작업 개요

개발 모듈(GITHUB_ISSUE_HELPER, TRANSLATOR)에 대한 버전관리 기능 전면 삭제. changelog가 별도로 관리되지 않아 해당 로직을 제거하여 코드베이스 정리.

**이슈**: [#153](https://github.com/Cassiiopeia/suh-project-utility/issues/153)

---

### 🎯 구현 목표

- `Suh-Domain-Module` 전체 모듈 삭제
- 버전관리 관련 엔티티, 서비스, 리포지토리 삭제
- HTML 페이지에서 버전 정보 UI 제거
- Gradle 의존성 정리

---

### ✅ 구현 내용

#### 1. Suh-Domain-Module 전체 모듈 삭제
- **삭제 대상**: `Suh-Domain-Module/` 폴더 전체
- **포함 파일**:
  - `dto/ModuleVersionRequest.java`
  - `dto/ModuleVersionResponse.java`
  - `service/ModuleVersionService.java`
  - `test/ModuleVersionServiceTest.java`
  - `build.gradle`
- **이유**: 버전관리 기능 미사용으로 전체 모듈 불필요

#### 2. Suh-Common 버전관리 관련 파일 삭제
- **파일**: `Suh-Common/src/main/java/me/suhsaechan/common/`
- **삭제 목록**:
  - `constant/ModuleType.java` - 모듈 타입 enum
  - `constant/ModuleUpdateType.java` - 업데이트 타입 enum
  - `entity/ModuleVersion.java` - 버전 엔티티
  - `entity/ModuleVersionUpdate.java` - 업데이트 엔티티
  - `repository/ModuleVersionRepository.java`
  - `repository/ModuleVersionUpdateRepository.java`
- **이유**: 버전관리 데이터 모델 및 저장소 불필요

#### 3. ModuleVersionController 삭제
- **파일**: `Suh-Web/src/main/java/me/suhsaechan/web/controller/api/ModuleVersionController.java`
- **삭제된 엔드포인트**: `POST /api/module/get/versions`
- **이유**: 버전 정보 조회 API 불필요

#### 4. versionInfo 프래그먼트 삭제
- **파일**: `Suh-Web/src/main/resources/templates/fragments/versionInfo.html`
- **내용**: 버전 정보 아코디언 UI 컴포넌트 (약 430줄)
- **이유**: 더 이상 사용되지 않는 프래그먼트

#### 5. HTML 페이지 수정

##### translator.html
- **파일**: `Suh-Web/src/main/resources/templates/pages/translator.html`
- **변경 내용**: 버전 정보 섹션 제거
- **삭제된 코드**:
  ```html
  <!-- 버전 정보 섹션 -->
  <div th:replace="~{fragments/versionInfo :: versionInfo(moduleType='TRANSLATOR')}"></div>
  ```

##### githubIssueHelper.html
- **파일**: `Suh-Web/src/main/resources/templates/pages/githubIssueHelper.html`
- **변경 내용**: 버전 정보 섹션 제거
- **삭제된 코드**:
  ```html
  <!-- 버전 정보 섹션 -->
  <div th:replace="~{fragments/versionInfo :: versionInfo(moduleType='GITHUB_ISSUE_HELPER')}"></div>
  ```

#### 6. Gradle 설정 수정

##### settings.gradle
- **변경 내용**: `include 'Suh-Domain-Module'` 라인 삭제
- **이유**: 삭제된 모듈 참조 제거

##### Suh-Web/build.gradle
- **변경 내용**: `implementation project(':Suh-Domain-Module')` 의존성 삭제
- **이유**: 삭제된 모듈 의존성 제거

##### Suh-Application/build.gradle
- **변경 내용**: `implementation project(':Suh-Domain-Module')` 의존성 삭제
- **이유**: 삭제된 모듈 의존성 제거

### 📌 참고사항

- **데이터베이스**: `module_version`, `module_version_update` 테이블은 코드에서 더 이상 사용되지 않음. 필요시 수동으로 테이블 삭제 가능
- **영향 범위**: 번역기, GitHub 이슈 헬퍼 기능 자체는 정상 동작. 버전 정보 UI만 제거됨
- **하위 호환성**: 고려하지 않음 (이슈 요구사항에 따름)
