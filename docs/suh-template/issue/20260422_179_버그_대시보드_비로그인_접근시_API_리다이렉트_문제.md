# ❗[버그][대시보드] 비로그인 상태에서 대시보드 API 로그인 페이지 리다이렉트 문제

- **라벨**: 작업전
- **담당자**: Cassiiopeia

---

🗒️ 설명
---

- 대시보드(`/`, `/dashboard`)는 `permitAll`로 공개되어 있으나, 대시보드에서 호출하는 3개의 API가 인증 필요 상태로 남아 있어 비로그인 사용자 접근 시 로그인 페이지로 302 리다이렉트 발생
- 결과적으로 서버 통계, 공지사항, Task D-Day 섹션이 모두 빈 화면 또는 에러 상태로 표시됨

🔄 재현 방법
---

1. 로그아웃 상태에서 `https://lab.suhsaechan.kr/` 접속
2. 브라우저 개발자 도구 Network 탭 확인
3. `/api/dashboard/summary`, `/api/notice/get/active`, `/api/task/goal/list` 요청이 302로 `/login`으로 리다이렉트됨

📸 참고 자료
---

- Network 탭에서 `summary`, `active`, `list` 요청 모두 Status 302 확인

✅ 예상 동작
---

- 비로그인 사용자도 대시보드에서 서버 통계, 공지사항, Task D-Day 목록을 정상적으로 조회할 수 있어야 함

⚙️ 환경 정보
---

- **OS**: macOS
- **브라우저**: Chrome
- **기기**: Desktop
