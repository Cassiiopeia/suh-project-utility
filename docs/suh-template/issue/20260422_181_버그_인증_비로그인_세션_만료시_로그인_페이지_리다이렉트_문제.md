# ❗[버그][인증] 비로그인 상태에서 세션 만료 시 로그인 페이지로 리다이렉트되는 문제

- **라벨**: 작업전
- **담당자**: Cassiiopeia

---

🗒️ 설명
---

- `invalidSessionUrl`이 `/login?sessionExpired=true`로 설정되어 있어, 비로그인 사용자가 대시보드 접근 시 세션이 없거나 만료된 경우 로그인 페이지로 강제 리다이렉트됨
- 대시보드(`/`)는 `permitAll` 공개 경로임에도 불구하고 세션 만료 정책으로 인해 비로그인 사용자가 정상 접근 불가

🔄 재현 방법
---

1. 로그아웃 또는 비로그인 상태에서 브라우저 캐시/세션 초기화
2. `https://lab.suhsaechan.kr/` 접속
3. `/login?sessionExpired=true` 로그인 페이지로 리다이렉트됨

📸 참고 자료
---

- 리다이렉트 URL: `/login?sessionExpired=true`
- 로그인 페이지 노출 (Welcome To SUH-LAB 화면)

✅ 예상 동작
---

- 비로그인 사용자가 `/` 접속 시 세션 유무와 관계없이 대시보드 페이지가 정상 표시되어야 함
- `invalidSessionUrl`을 `/`로 변경하여 비로그인 세션 만료 시 대시보드로 이동

⚙️ 환경 정보
---

- **OS**: macOS
- **브라우저**: Chrome
- **기기**: Desktop
