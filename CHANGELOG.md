## [0.0.4] - 2025-08-12

**PR:** #59  

**신규 기능**
- CI 로그에 최신 버전 표시(Updated Version 출력).

**버그 수정**
- 오타로 잘못된 API 경로(commmit → commit) 전반 수정(백엔드/템플릿/공개 엔드포인트).

- 버전 관리 워크플로우가 변경 사항을 일괄 스테이징하고 조건부 커밋.
- 스프링 프로젝트의 버전 동기화·다중 모듈 업데이트 개선.
- 프로젝트 버전 0.0.4로 업데이트.

**문서**
