## [1.1.5] - 2025-09-02

**PR:** #80  

**New Features**
- 스터디 모듈 추가: 카테고리/포스트/첨부파일 관리 API와 페이지(메인, 카테고리, 상세, 폼), 첨부 업로드/목록/삭제 엔드포인트.
- 공지 댓글 API 추가: 목록/생성/삭제.
- 대시보드 “스터디 플랜” 카드 및 헤더 메뉴 추가.

**Refactor**
- 공지 도메인 DTO 통합 및 패키지 정리, Docker 컨트롤러 엔드포인트 명칭 정리.

**Style**
- 스터디 전용 UI 스타일(common.css) 대거 추가, 에디터/마크다운/하이라이트 CSS 포함.

**Documentation**
- CLAUDE.md 가이드 추가, 코드 스타일 가이드 업데이트.

**Chores**
- 버전 1.1.5로 상향, 정적 업로드 경로 공개 설정, CI 워크플로우 유연화 및 권한/옵션 추가.

## [0.0.11] - 2025-08-15

**PR:** #71  

**신기능**
- 공지 댓글 기능 추가: 목록 조회·작성·삭제 지원, 본인 작성 댓글만 삭제 가능.
- 공지 상세 모달에 댓글 섹션·작성 폼 및 자동 새로고침 연동.
- 클라이언트 해시 조회 API 추가 및 프론트에서 세션 저장.

**스타일**
- 댓글 UI 모던 카드형으로 개선, 반응형 최적화.
- Docker 로그 화면 가독성 향상(색상, 스켈레톤, 상태 아이콘).

**문서**
- 코드 스타일 가이드 REST 섹션 갱신, README 버전 표기 업데이트.

**작업**
- 버전 0.0.11으로 상향.
